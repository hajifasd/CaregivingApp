如何获取用户ID和护工ID？

方法1: 从localStorage获取

  1. 在用户端页面按F12打开开发者工具
  2. 切换到Application/应用程序标签
  3. 在左侧找到Local Storage > http://localhost:8000
  4. 找到user_token，复制完整内容
  5. 在护工端页面同样操作，找到caregiver_token

方法2: 从控制台获取

  1. 在用户端页面按F12打开开发者工具
  2. 切换到Console/控制台标签
  3. 输入: localStorage.getItem('user_token')
  4. 复制输出的token内容
  5. 在护工端页面同样操作，输入: localStorage.getItem('caregiver_token')

方法3: 直接查看页面源码

  1. 在用户端页面按F12打开开发者工具
  2. 切换到Console/控制台标签
  3. 输入: getCurrentUserId()
  4. 复制输出的用户ID
  5. 在护工端页面同样操作，输入: getCurrentCaregiverId()
